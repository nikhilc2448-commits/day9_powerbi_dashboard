# Day 9 - Power BI Dashboard

## Objective

This project demonstrates the creation of an interactive Power BI dashboard by combining customer and order data. It includes data cleaning, data modeling, DAX measures, KPI cards, interactive slicers, and multiple visualizations to analyze sales performance.

## Dataset

- Customers.csv
- Orders.csv

## Features

- Imported and cleaned data using Power Query
- Created relationship between Customers and Orders using CustomerID
- Developed DAX measures
- Interactive KPI cards
- Sales by Category bar chart
- Sales by Month line chart
- Orders by Segment pie chart
- Customer sales summary table
- Region and Category slicers for filtering

## KPI Metrics

- Total Revenue
- Total Profit
- Order Count
- Profit Margin %

## Tools Used

- Microsoft Power BI Desktop
- Power Query
- DAX

## Dashboard Insights

- Electronics generated the highest sales among all product categories.
- Consumer and Corporate segments contributed the largest share of orders.
- Sales fluctuated across different dates, showing noticeable peaks during the month.
- The dashboard allows users to filter data dynamically by Region and Category for better analysis.
